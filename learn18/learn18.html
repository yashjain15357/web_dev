Box Shadow syntax: box-shadow: h-offset v-offset blur spread color | none | inset | initial | inherit;
It sets the horizontal position of the shadow. Positive values position the shadow to the right, and negative values position it to the left.
It sets the vertical position of the shadow. Positive values position the shadow at the bottom, and negative values position it at the top.
It sets the amount of blur in the shadow.
It sets the color of the shadow.
It is used to set the size of the shadow. The size of the spread depends on the value of the spread.
Using the inset, we can create the shadow inside the box.
